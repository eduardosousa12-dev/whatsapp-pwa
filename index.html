WhatsApp Web 2

Use uma segunda conta no celular

1 Ative o Modo Desktop

O WhatsApp Web só funciona em modo desktop. Selecione seu navegador:

  1. Toque no menu ⋮ (3 pontinhos)
  2. Marque "Site para desktop"
  3. A página vai recarregar
  1. Toque em "aA" na barra de endereço
  2. Selecione "Solicitar Site para Desktop"
  1. Toque no menu ☰ (3 linhas)
  2. Ative "Site para desktop"
2 Como conectar

Após abrir o WhatsApp Web, pegue o outro celular (com a conta que quer conectar) e vá em:

WhatsApp → Configurações → Dispositivos conectados → Conectar dispositivo

Escaneie o QR code que aparecer na tela.

⚠️ Você precisa de OUTRO celular com WhatsApp para escanear o QR code. Este app serve para espelhar uma conta existente.

Marque a opção acima confirmando que ativou o modo desktop para liberar o botão.
